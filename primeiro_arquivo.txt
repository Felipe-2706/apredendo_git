O git é uma ferramenta essencial para o desenvolvimento de um projeto, nele podemos salvar o arquivo diversas vezes e sempre podemos ver as alterações realizadas por ultimo, sem contar que trabalhando em equipe. Podemos ver a ultima atualização tambem do parceiro e verificar se está correta e um c´codigo legivel. 
Os comandos executados mais vezes seriam para vizualizar os status, as mudanças e o historico. Lembrando que o Git possui 3 estados o Committed que significa que os dados estão sendo armazenados de forma segura, o modificadoa que siginifica que arquivos foram alterados e o  preparado que siginifica que está pronto para ser salvo como a versão atual do seu arquivo.
